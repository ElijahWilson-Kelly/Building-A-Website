the skeleton of your website

Getting Started

Before we start making a website we have to decide what kind of information we want to display. Do we want a heading? maybe more than one? maybe a few with different levels of importance? how about some plain text and an image and if we are feeling really crazy a video as well. To do this we need HTML. HTML or (HyperText Markup Language) is used to structure our websites and can be thought of as the skeleton of our website. It gives form and structure but can leave something to be desired when it comes to the looks department. But we have to start somewhere so let's write some HTML code.

Elements

A good place to start is to give our page a heading. But not just any heading, we want this heading to put all other headings on the page to shame. What we want to do is wrap the text we want to show in a HTML element. In this case an h1 element. To do this we write it like so.

<h1>My Awesome Website</h1>

Those red items before and after the text are called "tags" and often, as is the case here, come in pairs. To create an opening tag you first write a less than sign (<) followed by the element name (in this case h1) followed by a greater than sign (>).

Now if the element needs a closing tag you need to add that at the end. The closing tag is the same as the opening tag just with a forward slash (/) before the element name. So it looks like </h1>.

We don't just have to make headings. After all that would make a rather boring website. We can also add paragraphs, images, quotes, videos and a whole lot more.

Bellow are a few common element tags to get you started. There are many more than these and if you are interested here is a resource for you to check out the whole list of options. But for now these will do.

Common Tags

<h1-h6></h1-h6> - The heading tags. These are number from 1 to 6 from most important to least. It is generally recommended to only use one h1 tag per page
<p></p> - The paragraph tags. Used for writing paragraphs.
<img /> - The image tag. This is a self closing tag so does not need another tag to close it. Used to add images to the site.
<a></a> - The anchor tags. Used to link to other parts of the page or other websites.
<ul></ul> - The unordered list tags. Used to wrap around li elements to create an unordered list.
<ol></ul> - The ordered list tags. Used to wrap around li elements to create an ordered list.
<li></li> - The list item tags. Used within either the ul or ol elements to create a list item.

So now we are ready to add some more stuff to our website. Lets go.

<h1>My Awesome Website</h1>
<p>Welcome to my awesome website. Thanks for stopping in. I hope you will enjoy yourself!</p>
<h2>My Favorite Films</h2>
<ol>
<li>Lord of the Rings</li>
<li>Star Wars</li>
<li>Avatar</li>
</ol>

Well... I don't know about you but I think our website is really coming along. But what if we want to add some images to really spice it up. Well to do that we will have to learn about HTML attributes

Attributes

We can also add attributes to our HTML elements. We do this by adding a name and value before the end of the opening tag. Like so

<h1 class="important" >I AM A HEADING!</h1>

Here we've added a class attribute of "important" to the h1 heading. (This will be super useful when we start styling with CSS later on).

We can also add attributes to img and a elements to allow us to use images and link to other pages using the src and href attributes respectivly

<img src="mycutecatphoto.png" />
<a href="www.information.com" >More Info</a>

So lets add some images and links to our website

<h1>My Awesome Website</h1>
<p>Welcome to my awesome website. Thanks for stopping in. I hope you will enjoy yourself!</p>
<h2>My Favorite Films</h2>
<ol>
<li>Lord of the Rings</li>
<li>Star Wars</li>
<li>Avatar</li>
</ol>
<img src="mycutecatphoto.jpg" />
<p>Thanks for checking out my page. You can <a href="www.contactme.com" >contact me</a> if you have any questions</p>

We also need to add a <!DOCTYPE html> decleration at the top to let the computer know its an html file and a html tag with a head and body tag inside. All of the code we have written so far goes into the body tag.

So after we have added everything our code will look like this.

<!DOCTYPE html>
<html>
<head>
</head>
<body>
<h1>My Awesome Website</h1>
<p>Welcome to my awesome website. Thanks for stopping in. I hope you will enjoy yourself!</p>
<h2>My Favorite Films</h2>
<ol>
<li>Lord of the Rings</li>
<li>Star Wars</li>
<li>Avatar</li>
</ol>
<img src="mycutecatphoto.jpg" />
<p>Thanks for checking out my page. You can <a href="www.contactme.com" >contact me</a> if you have any questions</p>
</body>
</html>

Well i think that should do for now. Shall we have a look at how it turned out?

Hmmm..... It looks a bit boring. Maybe we should dress it up a little. And to the rescue comes CSS.
