Getting Started

We now know how to get our information onto the page. But what if we really want our website to express our uniqueness. For that we can use CSS. CSS or Cascading StyleSheet is used to add flavor and styling to our website. We do this by selecting HTML elements and adding various styles to them.
You can write CSS in your HTML file (within style tags) but it is generally recommended to keep your CSS in a seperate file, in which case we have to link to it.
You can do that like so.

<link rel="stylesheet" href="mystyle.css" />

We put this tag into the head tag in our html. So with our link added our file will look like this.

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="mystyle.css" />
</head>
<body>
<h1>My Awesome Website</h1>
<p>Welcome to my awesome website. Thanks for stopping in. I hope you will enjoy yourself!</p>
<h2>My Favorite Films</h2>
<ol>
<li>Lord of the Rings</li>
<li>Star Wars</li>
<li>Avatar</li>
</ol>
<img src="mycutecatphoto.png" />
<p>Thanks for checking out my page. You can <a href="www.contactme.com" >contact Me</a> if you have any questions</p>
</body>
</html>

Awesome. Now we can move onto our "mystyle.css" file.

Selectors

Suppose we want our p elements to be a little bigger and maybe blue. One way we could do that is by selecting all the p elements in the document and apply some styles to them all. To do this we write the element name (in this case p) followed by a set of curly brackets ({}). We than add our styles inside the brackets by writing a property-name followed by a (:) then a value followed by a (;). Like so.

p {
color: blue;
font-size: 22px;
}

We can also select every element with a certain class. To do this we write the class name with a period (.) before it.

.red {
color: red;
font-style: italic;
}

Or we can select an item by its individual id. To do this we write the id name with a hashtag (#) before it.

#submit-button {
height: 20px;
width: 60px;
}

Common Properties

There are many properties you can use to style your website and if you are interested here is a resource for you to check out. But for now I will give you a few to get you started.

color - used to change the color of the text.
font-family - used to set the font of your text.
font-size - used to change the size of the text.
background-color - used to change the background color of the element.
border - used to create a border around the content.
padding - used to add space between the content and the border.
margin - used to add space between the selected element and other elements.
width - used to set the width of the element.
height - used to set the height of the element.

Box Model

When it comes to positioning your elements on the page it is useful to think of all your elements as being in a box. You have one box with your content in it. Then around that we have another box for some padding. Then we have a box for the border. Then lastly we have a box for the margin.
